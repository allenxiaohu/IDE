# IntelliJ IDEA For Mac 快捷键

在 IntelliJ IDEA 中有两个 Mac 版本的快捷键，一个叫做：Mac OS X，一个叫做：Mac OS X 10.5+，此版本是Mac OS X 10.5+，根据官方PDF进行翻译。目前都是用 Mac OS X 10.5+。[官方英文PDF](https://resources.jetbrains.com/storage/products/intellij-idea/docs/IntelliJIDEA_ReferenceCard.pdf)

[为什么会有两套快捷键](https://intellij-support.jetbrains.com/hc/en-us/community/posts/206159109-Updated-Mac-OS-X-keymap-Feedback-needed) 


## Mac键盘符号和修饰键说明

* <strong>⌘</strong>  Command
* <strong>⇧</strong> Shift
* <strong>⌥</strong> Option
* <strong>⌃</strong> Control
* <strong>↩</strong> Return/Enter
* <strong>⌫</strong> Delete
* <strong>⌦</strong> 向前删除键（Fn+Delete）
* <strong>↑</strong> 上箭头
* <strong>↓</strong> 下箭头
* <strong>←</strong> 左箭头
* <strong>→</strong> 右箭头
* <strong>⇞</strong> Page Up（Fn+↑）
* <strong>⇟</strong> Page Down（Fn+↓）
* <strong>Home</strong> Fn + ←
* <strong>End</strong> Fn + →
* <strong>⇥</strong> 右制表符（Tab键）
* <strong>⇤</strong> 左制表符（Shift+Tab）
* <strong>⎋</strong> Escape (Esc)

## Editing（编辑）

* <strong>Control + Space</strong> 基本的代码补全（补全任何类、方法、变量）
* <strong>Control + Shift + Space</strong> 智能代码补全（过滤器方法列表和变量的预期类型）
* <strong>Command + Shift + Enter</strong> 自动结束代码，行末自动添加分号
* <strong>Command + P</strong> 显示方法的参数信息
* <strong>Control + J</strong> 快速查看文档
* <strong>Shift + F1</strong> 查看外部文档（在某些代码上会触发打开浏览器显示相关文档）
* <strong>Command + 鼠标放在代码上</strong> 显示代码简要信息
* <strong>Command + F1</strong> 在错误或警告处显示具体描述信息
* <strong>Command + N, Control + Enter, Control + N</strong> 生成代码（getter、setter、构造函数、hashCode/equals,toString）
* <strong>Control + O</strong> 覆盖方法（重写父类方法）
* <strong>Control + I</strong> 实现方法（实现接口中的方法）
* <strong>Command + Option + T</strong> 包围代码（使用if..else, try..catch, for, synchronized等包围选中的代码）
* <strong>Command + /</strong> 注释/取消注释与行注释
* <strong>Command + Option + /</strong> 注释/取消注释与块注释
* <strong>Option + 方向键上</strong> 连续选中代码块
* <strong>Option + 方向键下</strong> 减少当前选中的代码块
* <strong>Control + Shift + Q</strong> 显示上下文信息
* <strong>Option + Enter</strong> 显示意向动作和快速修复代码
* <strong>Command + Option + L</strong> 格式化代码
* <strong>Control + Option + O</strong> 优化import
* <strong>Control + Option + I</strong> 自动缩进线
* <strong>Tab / Shift + Tab</strong> 缩进代码 / 反缩进代码
* <strong>Command + X</strong> 剪切当前行或选定的块到剪贴板
* <strong>Command + C</strong> 复制当前行或选定的块到剪贴板
* <strong>Command + V</strong> 从剪贴板粘贴
* <strong>Command + Shift + V</strong> 从最近的缓冲区粘贴
* <strong>Command + D</strong> 复制当前行或选定的块
* <strong>Command + Delete</strong> 删除当前行或选定的块的行
* <strong>Control + Shift + J</strong> 智能的将代码拼接成一行
* <strong>Command + Enter</strong> 智能的拆分拼接的行
* <strong>Shift + Enter</strong> 开始新的一行
* <strong>Command + Shift + U</strong> 大小写切换
* <strong>Command + Shift + ] / Command + Shift + [</strong> 选择直到代码块结束/开始
* <strong>Option + Fn + Delete</strong> 删除到单词的末尾
* <strong>Option + Delete</strong> 删除到单词的开头
* <strong>Command + 加号 / Command + 减号</strong> 展开 / 折叠代码块
* <strong>Command + Shift + 加号</strong> 展开所有代码块
* <strong>Command + Shift + 减号</strong> 折叠所有代码块
* <strong>Command + W</strong> 关闭活动的编辑器选项卡

## Search/Replace（查询/替换）

* <strong>Double Shift</strong> 查询任何东西
* <strong>Command + F</strong> 文件内查找
* <strong>Command + G</strong> 查找模式下，向下查找
* <strong>Command + Shift + G</strong> 查找模式下，向上查找
* <strong>Command + R</strong> 文件内替换
* <strong>Command + Shift + F</strong> 全局查找（根据路径）
* <strong>Command + Shift + R</strong> 全局替换（根据路径）
* <strong>Command + Shift + S</strong> 查询结构（Ultimate Edition 版专用，需要在Keymap中设置）
* <strong>Command + Shift + M</strong> 替换结构（Ultimate Edition 版专用，需要在Keymap中设置）

## Usage Search（使用查询）

* <strong>Option + F7 / Command + F7</strong> 在文件中查找用法 / 在类中查找用法
* <strong>Command + Shift + F7</strong> 在文件中突出显示的用法
* <strong>Command + Option + F7</strong> 显示用法

## Compile and Run（编译和运行）

* <strong>Command + F9</strong> 编译Project
* <strong>Command + Shift + F9</strong> 编译选择的文件、包或模块
* <strong>Control + Option + R</strong> 弹出 Run 的可选择菜单
* <strong>Control + Option + D</strong> 弹出 Debug 的可选择菜单
* <strong>Control + R</strong> 运行
* <strong>Control + D</strong> 调试
* <strong>Control + Shift + R, Control + Shift + D</strong> 从编辑器运行上下文环境配置

## Debugging（调试）

* <strong>F8</strong> 进入下一步，如果当前行断点是一个方法，则不进入当前方法体内
* <strong>F7</strong> 进入下一步，如果当前行断点是一个方法，则进入当前方法体内，如果该方法体还有方法，则不会进入该内嵌的方法中
* <strong>Shift + F7</strong> 智能步入，断点所在行上有多个方法调用，会弹出进入哪个方法
* <strong>Shift + F8</strong> 跳出
* <strong>Option + F9</strong> 运行到光标处，如果光标前有其他断点会进入到该断点
* <strong>Option + F8</strong> 计算表达式（可以更改变量值使其生效）
* <strong>Command + Option + R</strong> 恢复程序运行，如果该断点下面代码还有断点则停在下一个断点上
* <strong>Command + F8</strong> 切换断点（若光标当前行有断点则取消断点，没有则加上断点）
* <strong>Command + Shift + F8</strong> 查看断点信息

## Navigation（导航）

* <strong>Command + O</strong> 查找类文件
* <strong>Command + Shift + O</strong> 查找所有类型文件、打开文件、打开目录，打开目录需要在输入的内容前面或后面加一个反斜杠/
* <strong>Command + Option + O</strong> 前往指定的变量 / 方法
* <strong>Control + 方向键左 / Control + 方向键右</strong> 左右切换打开的编辑tab页
* <strong>F12</strong> 返回到前一个工具窗口
* <strong>Esc</strong> 从工具窗口进入代码文件窗口
* <strong>Shift + Esc</strong> 隐藏当前或最后一个活动的窗口，且光标进入代码文件窗口
* <strong>Command + Shift + F4</strong> 关闭活动run/messages/find/... tab
* <strong>Command + L</strong> 在当前文件跳转到某一行的指定处
* <strong>Command + E</strong> 显示最近打开的文件记录列表
* <strong>Option + 方向键左 / Option + 方向键右</strong> 光标跳转到当前单词 / 中文句的左 / 右侧开头位置
* <strong>Command + Option + 方向键左 / Command + Option + 方向键右</strong> 退回 / 前进到上一个操作的地方
* <strong>Command + Shift + Delete</strong> 跳转到最后一个编辑的地方
* <strong>Option + F1</strong> 显示当前文件选择目标弹出层，弹出层中有很多目标可以进行选择(如在代码编辑窗口可以选择显示该文件的Finder)
* <strong>Command + B / Command + 鼠标点击</strong> 进入光标所在的方法/变量的接口或是定义处
* <strong>Command + Option + B</strong> 跳转到实现处，在某个调用的方法名上使用会跳到具体的实现处，可以跳过接口
* <strong>Option + Space, Command + Y</strong> 快速打开光标所在方法、类的定义
* <strong>Control + Shift + B</strong> 跳转到类型声明处
* <strong>Command + U</strong> 前往当前光标所在方法的父类的方法 / 接口定义
* <strong>Control + 方向键下 / Control + 方向键上</strong> 当前光标跳转到当前文件的前一个/后一个方法名位置
* <strong>Command + ] / Command + [</strong> 移动光标到当前所在代码的花括号开始/结束位置
* <strong>Command + F12</strong> 弹出当前文件结构层，可以在弹出的层上直接输入进行筛选（可用于搜索类中的方法）
* <strong>Control + H</strong> 显示当前类的层次结构
* <strong>Command + Shift + H</strong> 显示方法层次结构
* <strong>Control + Option + H</strong> 显示调用层次结构
* <strong>F2 / Shift + F2</strong> 跳转到下一个/上一个突出错误或警告的位置
* <strong>F4 / Command + 方向键下</strong> 编辑/查看代码源
* <strong>Option + Home</strong> 显示到当前文件的导航条
* <strong>F3</strong> 选中文件/文件夹/代码行，添加/取消书签
* <strong>Option + F3</strong> 选中文件/文件夹/代码行，使用助记符添加/取消书签
* <strong>Control + 0...Control + 9</strong> 定位到对应数值的书签位置
* <strong>Command + F3</strong> 显示所有书签

## Refactoring（重构）

* <strong>F5</strong> 复制文件到指定目录
* <strong>F6</strong> 移动文件到指定目录
* <strong>Command + Delete</strong> 在文件上为安全删除文件，弹出确认框
* <strong>Shift + F6</strong> 重命名文件
* <strong>Command + F6</strong> 更改签名
* <strong>Command + Option + N</strong> 一致性
* <strong>Command + Option + M</strong> 将选中的代码提取为方法
* <strong>Command + Option + V</strong> 提取变量
* <strong>Command + Option + F</strong> 提取字段
* <strong>Command + Option + C</strong> 提取常量
* <strong>Command + Option + P</strong> 提取参数

## VCS/Local History（版本控制/本地历史记录）

* <strong>Command + K</strong> 提交代码到版本控制器
* <strong>Command + T</strong> 从版本控制器更新代码
* <strong>Option + Shift + C</strong> 查看最近的变更记录
* <strong>Control + C</strong> 快速弹出版本控制器操作面板

## Live Templates（动态代码模板）

* <strong>Command + Option + J</strong> 弹出模板选择窗口，将选定的代码使用动态模板包住
* <strong>Command + J</strong> 插入自定义动态代码模板

## General（通用）

* <strong>Command + 1...Command + 9</strong> 打开相应编号的工具窗口
* <strong>Command + S</strong> 保存所有
* <strong>Command + Option + Y</strong> 同步、刷新
* <strong>Control + Command + F</strong> 切换全屏模式
* <strong>Command + Shift + F12</strong> 切换最大化编辑器
* <strong>Option + Shift + F</strong> 添加到收藏夹
* <strong>Option + Shift + I</strong> 检查当前文件与当前的配置文件
* <strong>Control + `</strong> 快速切换当前的scheme（切换主题、代码样式等）
* <strong>Command + ,</strong> 打开IDEA系统设置
* <strong>Command + ;</strong> 打开项目结构对话框
* <strong>Shift + Command + A</strong> 查找动作（可设置相关选项）
* <strong>Control + Shift + Tab</strong> 编辑窗口标签和工具窗口之间切换（如果在切换的过程加按上delete，则是关闭对应选中的窗口）

## Other（一些官方文档上没有体现的快捷键）

* <strong>Command + Shift + 8</strong> 竖编辑模式
